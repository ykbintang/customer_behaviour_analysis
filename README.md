# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, including data loading, exploratory data analysis (EDA), data cleaning, SQL querying, dashboard development, and reporting. The goal is to extract actionable insights and present them through interactive visualizations and a structured report.

---

## 📂 Dataset

* Source: *Kaggle*
* Format: CSV
* Description:
  The dataset contains information about Customer demographics (Age, Gender, Location, Subscription Status), Purchase details (Item Purchased, Category, Purchase Amount, Season, Size, Color), Shopping behavior (Discount Applied, Promo Code Used, Previous Purchases, Frequency of Purchases, Review Rating, Shipping Type).

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI** (Dashboard visualization)
* **Gamma** (Presentation creation)
* **Jupyter Notebook / VS Code**

## 🔍 Steps

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Connected to SQL database for querying

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized data formats
* Fixed inconsistencies

### 3. Exploratory Data Analysis (EDA)

* Analyzed data distributions
* Identified trends and patterns
* Detected outliers
* Generated summary statistics

### 4. SQL Analysis

* Wrote queries to extract insights
* Performed aggregations, joins, and filtering
* Created views for analysis

### 5. Dashboard Development

* Built interactive dashboard in Power BI
* Included key metrics and KPIs
* Designed user-friendly visualizations

### 6. Reporting

* Summarized insights in a structured report
* Created presentation using Gamma

## ▶️ How to Run

### 1. Clone Repository

```bash
git clone https://github.com/ykbintang/customer_behavior_analysis.git
cd your-repo-name
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Python Analysis

```bash
jupyter notebook
```

or

```bash
python main.py
```

### 4. SQL Setup

* Import dataset into your SQL database
* Run provided SQL scripts

### 5. Open Dashboard

* Open `.pbix` file in Power BI Desktop
